# Stephen的微信小程序项目快速开发脚手架

> 本项目封装了微信小程序登陆逻辑,全局分享等功能,流程完善

> 本项目封装了网络请求框架逻辑,集成分页逻辑,loading是否展现,请求参数/返回数据log输出

> 本项目预置了一些开发工具类,二维码/条形码生成等,使用非常方便
